# アパレルショップ商品ページ模写
(セレクトボックス・テーブル実装)

## 概要
学習目的で制作した模写コーディング作品です。

## 使用技術
- HTML
- CSS

## 学んだこと意識したこと
- コードを書き始める前にサイトの全体のレイアウトを分析しパーツごとに分け、メモをとることから始める。
- デザインの共通点を見つけることでクラス名を付けやすくする。そうすることでコードの肥大化を防ぐこと。
-  情報を整理して伝えるため``<table>``タグを使って表を作成する。
-  ``<select>``,``<option>``を利用してユーザーが数値を直観的に選択出来るように構築する。
-  テーブル　表
